Learning platform

by Michal Skrzypek

I created this SPA from scratch so I could get more familiar with REST applications and the Angular framework. The application is based on Spring BOOT together with Spring Data and Spring Security. Moreover, JWT authentication was implemented to deal better with user authentication.

My GitHub account My LinkedIn account

Using the application as a regular user

First you need to create an user account. Then, after logging in, you will be able to look through all the courses available. Courses are sorted by the categories. You can add the course you are interested about to your courses. All of your courses can be seen in the "My courses" tab.

Using the application as an instructor

First you need to create an instructor account. Then, after logging in, you will be able to look through all the courses available AND ALSO add your unique course. It can be done by filling the form that you can find by pressing "Add new course!" button in the navbar. ONLY USERS WITH THE INSTRUCTOR ROLE have access to that form.
